# Electron + create-react-app
![](https://cdn-images-1.medium.com/max/1024/1*rI2NNwTLfX_TH3b7OE93JQ.png)

## Quick Start
```
# Clone
git clone git@github.com:jason-michael/electron-cra-starter.git

# Install
yarn

# Run
yarn dev
```

## Features
- Hot reload
- Font Awesome
- styled-components
- working `fs`

## Package
```
yarn electron-pack
```
